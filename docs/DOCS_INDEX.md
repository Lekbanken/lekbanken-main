## Placement Rules Summary

- Aktiva områdesdocs ska över tid flytta in i respektive klustermapp.
- Historisk områdesdocs ska över tid flytta djupare i samma kluster, till exempel `history/`, `archive/` eller annan låg-trust-yta.
- Root `docs/` ska successivt bli mer governance-first och mindre blandad katalog.
- Om du behöver full status eller klassificering: uppdatera [INVENTORY.md](INVENTORY.md), inte den här filen.
