# Prompts (archive)

Samling av prompt-filer för AI-assistenter och olika analyser/genereringar.

Syfte:
- Hålla `docs/` mer fokuserat på “source of truth” (domäner, runbooks, kontrakt)
- Fortfarande enkelt att hitta prompts vid behov

Regler:
- Prompts är **hjälptexter**, inte specifikationer.
- Om en prompt etablerar en stabil process, sammanfatta den i en riktig doc (t.ex. `docs/AI_CODING_GUIDELINES.md`) och länka hit.
