# Getting Started

This guide will help you get up and running with the Legendary Escape Room Toolkit in just a few minutes.

## Prerequisites

Before you begin, ensure you have:

- A modern web browser (Chrome, Firefox, Safari, Edge)
- An account with administrator access
- At least one escape room game idea in mind

## Quick Start

### 1. Log In to Admin Dashboard

Navigate to your toolkit URL and log in with your credentials:

```
https://your-domain.com/admin
```

### 2. Create Your First Game

1. Click **"Create Game"** in the dashboard
2. Enter a name and description for your game
3. Set the time limit (e.g., 60 minutes)
4. Click **"Create"**

### 3. Add Content

#### Phases

Phases are major sections of your game:

1. Go to the **Phases** tab
2. Click **"Add Phase"**
3. Name it (e.g., "Introduction", "Main Challenge", "Finale")

#### Steps

Steps are individual content items within phases:

1. Select a phase
2. Click **"Add Step"**
3. Choose a step type:
   - **Narrative**: Story text
   - **Puzzle**: Interactive challenge
   - **Reveal**: Clue or artifact reveal
   - **Decision**: Voting/choice point

### 4. Start a Session

1. From the game detail page, click **"Start Session"**
2. Share the session code with participants
3. Wait for participants to join
4. Click **"Begin"** when ready

### 5. Run the Game

- Use the host dashboard to monitor progress
- Advance steps manually or let triggers automate
- Use the time bank for hints and penalties
- End the session when complete
