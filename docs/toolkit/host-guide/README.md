## Quick Reference

### Session Controls

| Action | Description | Keyboard |
|--------|-------------|----------|
| Start | Begin the session | `Space` |
| Pause | Pause timer | `P` |
| Resume | Resume timer | `Space` |
| Next Step | Advance to next step | `→` |
| Previous Step | Go back one step | `←` |
| End Session | Complete the game | `End` |

## Best Practices

### Before the Session

- [ ] Test your game with a practice run
- [ ] Verify all triggers are working
- [ ] Check participant capacity settings
- [ ] Prepare any physical props needed
- [ ] Have backup plans for technical issues

### During the Session

- [ ] Monitor participant connections
- [ ] Watch for stuck participants
- [ ] Use time bank for hints/penalties
- [ ] Keep an eye on the timeline
- [ ] Be ready to intervene if needed

### After the Session

- [ ] Review session analytics
- [ ] Note any issues for improvement
- [ ] Archive completed sessions
- [ ] Gather participant feedback

## Common Scenarios

### Participant Can't Connect

1. Verify they're using the correct session code
2. Check if session is active
3. Ask them to refresh the page
4. Try a different browser
5. Use incognito/private mode

### Timer Issues

1. Pause the session if needed
2. Use time bank to adjust
3. Manual time adjustments available in settings

### Trigger Not Firing

1. Check trigger conditions in game builder
2. Verify signal names match exactly
3. Check trigger is enabled
4. Review trigger logs in analytics
